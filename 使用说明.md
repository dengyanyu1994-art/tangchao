# 唐朝宫廷MVU系统使用说明

## 一、系统概述

唐朝宫廷MVU系统是一个专为SillyTavern设计的变量管理系统，支持：
- MVU（Model-View-Update）架构
- 4阶段剧情系统
- 角色关系管理
- 变量自动更新

## 二、部署方法

### 方法1：GitHub + jsDelivr部署

1. 将 `bundle.js` 文件上传到GitHub仓库
2. 使用jsDelivr CDN链接：
   ```
   https://testingcf.jsdelivr.net/gh/你的用户名/你的仓库名@main/bundle.js
   ```

### 方法2：本地部署

1. 将 `bundle.js` 文件放到SillyTavern的扩展目录
2. 在SillyTavern设置中启用该扩展

## 三、核心功能

### 3.1 变量管理

#### 获取变量
```javascript
// 获取主角声望值
const reputation = TangCourtMVU.system.getVariable('protagonist_reputation');
console.log('当前声望:', reputation);

// 获取当前时代
const era = TangCourtMVU.system.getVariable('world_era');
console.log('当前时代:', era);
```

#### 设置变量
```javascript
// 设置主角声望值
TangCourtMVU.system.setVariable('protagonist_reputation', 50);

// 设置当前时代
TangCourtMVU.system.setVariable('world_era', '贞观');
```

#### 增加变量
```javascript
// 增加声望值
TangCourtMVU.system.incrementVariable('protagonist_reputation', 10);
```

### 3.2 角色关系管理

#### 获取角色关系
```javascript
// 获取长乐公主的关系数据
const relationship = TangCourtMVU.system.getRelationship('princess_changle');
console.log('好感度:', relationship.affection);
console.log('信任度:', relationship.trust);
console.log('关系阶段:', relationship.relationship_stage);
```

#### 更新角色关系
```javascript
// 更新长乐公主的好感度和信任度
TangCourtMVU.system.updateRelationship('princess_changle', {
  affection: 45,
  trust: 30,
  interaction_count: 8
});
```

### 3.3 四阶段剧情系统

#### 阶段定义
- **阶段1 - 初识阶段**：好感度0-30，信任度0-20，互动0-5次
- **阶段2 - 熟悉阶段**：好感度31-60，信任度21-50，互动6-15次
- **阶段3 - 亲密阶段**：好感度61-85，信任度51-80，互动16-30次
- **阶段4 - 羁绊阶段**：好感度86-100，信任度81-100，互动31次以上

#### 获取阶段信息
```javascript
// 获取阶段1的详细信息
const stageInfo = TangCourtMVU.system.getStageInfo(1);
console.log('阶段名称:', stageInfo.name);
console.log('可用行动:', stageInfo.available_actions);
console.log('锁定行动:', stageInfo.locked_actions);
```

#### 执行行动
```javascript
// 执行"问候"动作
const result = TangCourtMVU.system.performAction('princess_changle', '问候');
if (result.success) {
  console.log('行动成功:', result.message);
} else {
  console.log('行动失败:', result.message);
}
```

### 3.4 角色信息查询

```javascript
// 获取长乐公主的详细信息
const characterInfo = TangCourtMVU.system.getCharacterInfo('princess_changle');
console.log('姓名:', characterInfo.name);
console.log('年龄:', characterInfo.age);
console.log('外貌:', characterInfo.appearance);
console.log('喜好礼物:', characterInfo.preferences.liked_gifts);
```

### 3.5 数据导入导出

#### 导出数据
```javascript
const exportedData = TangCourtMVU.system.exportVariables();
console.log('导出的数据:', exportedData);
// 可以保存到本地存储或文件
localStorage.setItem('tangCourtData', exportedData);
```

#### 导入数据
```javascript
// 从本地存储读取
const savedData = localStorage.getItem('tangCourtData');
if (savedData) {
  TangCourtMVU.system.importVariables(savedData);
  console.log('数据导入成功');
}
```

## 四、事件监听

### 4.1 变量变更事件
```javascript
TangCourtMVU.system.on('variableChanged', (data) => {
  console.log(`变量 ${data.key} 已更新为 ${data.value}`);
});
```

### 4.2 关系更新事件
```javascript
TangCourtMVU.system.on('relationshipUpdated', (data) => {
  console.log(`角色 ${data.characterId} 的关系已更新`);
  console.log('新关系数据:', data.relationship);
});
```

### 4.3 阶段转换事件
```javascript
TangCourtMVU.system.on('stageTransition', (data) => {
  console.log(`角色 ${data.characterId} 从阶段 ${data.oldStage} 转换到阶段 ${data.newStage}`);
});
```

### 4.4 行动执行事件
```javascript
TangCourtMVU.system.on('actionPerformed', (data) => {
  console.log(`对角色 ${data.characterId} 执行了动作: ${data.action}`);
});
```

## 五、完整示例

### 5.1 初次见面场景
```javascript
// 初始化长乐公主的关系
const relationship = TangCourtMVU.system.getRelationship('princess_changle');

// 执行问候动作
TangCourtMVU.system.performAction('princess_changle', '问候');

// 更新关系数据
TangCourtMVU.system.updateRelationship('princess_changle', {
  affection: 10,
  trust: 5
});

// 检查当前阶段
console.log('当前阶段:', relationship.relationship_stage);
```

### 5.2 赠送礼物场景
```javascript
// 获取角色信息
const characterInfo = TangCourtMVU.system.getCharacterInfo('princess_changle');

// 检查礼物是否受喜欢
const gift = '诗词集';
const isLiked = characterInfo.preferences.liked_gifts.includes(gift);

if (isLiked) {
  // 执行赠礼动作
  TangCourtMVU.system.performAction('princess_changle', '赠送礼物');
  
  // 增加好感度
  TangCourtMVU.system.updateRelationship('princess_changle', {
    affection: relationship.affection + 15,
    trust: relationship.trust + 5
  });
  
  console.log('长乐公主很喜欢你的礼物！');
} else {
  console.log('长乐公主对这个礼物不太感兴趣...');
}
```

### 5.3 阶段转换检测
```javascript
// 监听阶段转换
TangCourtMVU.system.on('stageTransition', (data) => {
  const stageInfo = TangCourtMVU.system.getStageInfo(data.newStage);
  
  console.log(`=== 关系升级 ===`);
  console.log(`进入 ${stageInfo.name}`);
  console.log(`描述: ${stageInfo.description}`);
  console.log(`解锁行动: ${stageInfo.available_actions.join(', ')}`);
  
  // 显示通知
  alert(`恭喜！你与长乐公主的关系进入了"${stageInfo.name}"！`);
});

// 触发阶段转换
TangCourtMVU.system.updateRelationship('princess_changle', {
  affection: 35,
  trust: 25,
  interaction_count: 8
});
```

## 六、变量参考

### 6.1 世界变量
- `world_era`: 当前时代（武德/贞观/永徽/武周/开元/天宝）
- `world_year`: 当前年份（618-755）
- `world_month`: 当前月份（1-12）
- `world_day`: 当前日期（1-30）
- `world_season`: 当前季节（春/夏/秋/冬）
- `world_time_period`: 当前时段（早朝/午时/午后/黄昏/夜晚/深夜）

### 6.2 主角变量
- `protagonist_name`: 主角姓名
- `protagonist_gender`: 主角性别
- `protagonist_age`: 主角年龄
- `protagonist_identity`: 主角身份
- `protagonist_reputation`: 主角声望值（-100到100）
- `protagonist_court_influence`: 朝堂影响力（0-100）
- `protagonist_harem_status`: 后宫地位（0-100）
- `protagonist_political_power`: 政治权力（0-100）
- `protagonist_imperial_favor`: 帝王宠爱（0-100）
- `protagonist_family_influence`: 家族势力（0-100）
- `protagonist_wealth`: 财富（0-10000）

### 6.3 角色关系变量
- `affection`: 好感度（0-100）
- `trust`: 信任度（0-100）
- `intimacy`: 亲密度（0-100）
- `relationship_stage`: 关系阶段（1-4）
- `interaction_count`: 互动次数
- `last_interaction`: 最后互动时间
- `relationship_status`: 关系状态

## 九、注意事项

1. **变量范围**：所有数值变量都有最小值和最大值限制，超出范围会自动调整
2. **阶段转换**：阶段转换是自动的，当满足条件时会自动触发
3. **行动限制**：某些行动在特定阶段会被锁定，无法执行
4. **数据持久化**：建议定期导出数据并保存，避免数据丢失
5. **性能优化**：大量角色关系可能会影响性能，建议控制在50个以内

## 十、常见问题

### Q1: 如何添加新角色？
A: 在 `CHARACTER_DATABASE` 对象中添加新的角色信息即可。

### Q2: 如何自定义阶段要求？
A: 修改 `STORY_STAGES` 对象中的 `requirements` 配置。

### Q3: 如何保存游戏进度？
A: 使用 `exportVariables()` 导出数据，保存到本地存储或文件。

### Q4: 如何恢复游戏进度？
A: 使用 `importVariables()` 导入之前保存的数据。
